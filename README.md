# local_manifests
local manifests I use to build roms

## usage

```
cd .repo
git clone https://github.com/JuanCldCmt/local_manifests.git
```

In build/make/target/product/handheld_product.mk
```xml
    FakeStore \
    GmsCore \
    GsfProxy \
    MozillaNlpBackend \
    NominatimNlpBackend \
    FDroid \
    F-DroidPrivilegedExtension
```
